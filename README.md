# GeoVision AI

GeoVision AI is a beginner-friendly Python project for working with geospatial and image-based data.

## Project Structure

- `app/` - main application code
- `geo/` - geospatial processing code
- `data/raw/` - original data files
- `data/processed/` - cleaned or processed data
- `data/labels/` - labels or annotations
- `models/` - trained models
- `notebooks/` - Jupyter experiments
- `tests/` - test files
- `config/` - settings and configuration
